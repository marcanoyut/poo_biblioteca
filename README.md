# poo_biblioteca
Trabajo Practico

Sistema de Gestión de Biblioteca, para representar
entidades del mundo real, identificar responsabilidades, definir relaciones entre objetos y construir
una solución coherente, modular y extensible.

Objetivos 


A- analizar un problema del mundo real y traducirlo a un modelo orientado a objetos,
B- identificar clases, atributos, métodos y relaciones entre objetos,
C- aplicar principios de encapsulamiento, herencia, asociación, agregación o composición,
D- diseñar un sistema modular, mantenible y escalable,
E- representar el sistema mediante diagramas, 
F- implementar una versión funcional del modelo diseñado,
G- documentar y justificar las decisiones adoptadas.

Requerimiento y Analisis de Disenho

identificación del problema y sus actores,
relevamiento de requerimientos funcionales básicos,
identificación de clases del dominio,
definición de atributos y métodos,
determinación de relaciones entre clases,
distribución de responsabilidades,
diseño de una solución orientada a objetos justificada.
